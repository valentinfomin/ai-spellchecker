Privacy Policy for AI Spellchecker

Last Updated: January 30, 2026

This Privacy Policy explains how AI Spellchecker ("the Extension") handles user information. Our core principle is that your data belongs to you.

1. Data Collection and Transmission

AI Spellchecker is designed with a "local-first" architecture:

  • No Data Collection: We do not collect, store, or transmit any personally identifiable information (PII), browsing history, or text content to external servers.
  • Local Processing: All text analysis and grammar corrections are performed locally on your device using WebGPU and WebAssembly technology.
  • No Cloud Storage: Your text is never stored in any cloud database by this extension.

2. Permissions Usage

To function correctly, the extension requires the following permissions:

  • activeTab & host_permissions: Used solely to read the text you highlight and insert the corrected version back into the input field.
  • storage: Used only to cache AI model weights on your local machine to improve performance and avoid repeated downloads.
  • contextMenus: Used to provide the "Fix grammar (AI)" option in your browser's right-click menu.

3. Third-Party Services

The Extension uses the WebLLM library to run AI models locally. No data is shared with any third-party AI providers (such as OpenAI or Google) during the correction process.

4. Data Security

Since no data is transmitted or stored remotely, there is no risk of your data being intercepted from our servers. We rely on the native security sandboxing provided by the Chrome browser to protect the local execution environment.

5. Compliance

This policy complies with the Google Chrome Web Store Developer Program Policies regarding User Data.
